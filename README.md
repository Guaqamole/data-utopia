# data-utopia

집에서 직접 구축한 Data Platform, K8s, 대용량 데이터를 활용하여
- 실시간 데이터 수집
- 대용량 로그 수집
- 대용량 데이터 처리
- MLOps 
- CI/CD
- DL 프로젝트

등 나만의 데이터센터를 구축하고 데이터가 잘 활용 될 수 있도록, `Data Utopia`를 만들어간 자료들입니다.
